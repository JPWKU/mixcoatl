Install
-------

Mixcoatl is designed to be simple to install and update on many platforms. Mixcoatl can be installed on \*nix systems as well as windows and should work wherever python is available.

Mixcoatl has been tested on the following platforms:

* Windows 2008

* Windows 2012

* Ubuntu >= 12.04

* CentOS >= 6

* RHEL >= 6

* OS X
