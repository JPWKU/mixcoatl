classmethod add_snapshot(volume_id, name, description, budget, callback=None)[source]¶

Creates a snapshot from volume_id

Warning

Snapshot creation is an asynchronous task. Specifying a callback will cause a blocking operation while the snapshot completes. When using the callback, execution could block for a VERY long time depending on the time it takes to make the snapshot.

Parameters:
  • volume_id (int.) – The volume to snapshot
  • name (str.) – The name for the snapshot
  • description (str.) – Description of the snapshot
  • budget (int.) – The billing code for the snapshot
  • callback (func.) – An optional callback to send the final Snapshot.
Returns:

Snapshot
Raises:
